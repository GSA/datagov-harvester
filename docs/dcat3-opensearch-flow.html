DCAT-US 3.0 Harvest → OpenSearch Index Flow

Function call stack and data flow for schema_type = dcatus3.0 sources. DCAT3 records skip MDTranslator (ISO-only); raw JSON is validated, stored on Dataset.dcat, then transformed at index time into canonical search fields.

End-to-end flow (harvest job)

flowchart TD
  START([harvest_job_starter
harvester/harvest.py]) --> HS[HarvestSource.__post_init__]
  HS --> SCHEMA[Load dcatus3.0 schema + build_dcatus3_validator]
  SCHEMA --> RFH[HarvestSource.run_full_harvest]

  RFH --> ADS[acquire_data_sources]
  ADS --> INT[acquire_minimum_internal_data
→ store_records_as_internal]
  ADS --> EXT[acquire_minimum_external_data]

  EXT --> DL[download_file url .json
→ external_records = dataset array]
  DL --> FILT[filter_datasets_with_no_identifier
filter_duplicate_identifiers]
  FILT --> DEL[determine_internal_deletions
→ iter_internal_records_to_be_deleted]
  FILT --> ERP[external_records_to_process
yields Record per dataset]

  ERP --> SORT[sort_dataset record
json.dumps → source_raw]
  SORT --> REC[Record instance
identifier + metadata_hash]

  DEL --> HARVEST
  REC --> HARVEST[Record.harvest]

  HARVEST --> CMP[compare
create | update | skip]
  CMP -->|action is None| SKIP([skip record])
  CMP -->|create/update| VAL[validate
json.loads source_raw
validator.iter_errors]

  VAL -->|invalid| ERR([status=error, no sync])
  VAL -->|valid| SYNC[sync]

  SYNC --> META[_metadata_for_dataset
returns parsed DCAT3 dict]
  META --> PAYLOAD[_dataset_payload
dcat, slug, spatial, org, ...]
  PAYLOAD --> DB{action?}
  DB -->|create| INS[insert_dataset_with_unique_slug
→ Dataset row]
  DB -->|update| UPS[db_interface.upsert_dataset]
  DB -->|delete| DELDB[delete_dataset_by_slug
→ delete_dataset_by_id]

  INS --> IDX[_index_dataset_in_opensearch]
  UPS --> IDX
  DELDB --> OSDEL[_delete_dataset_from_opensearch]

  IDX --> OSINIT[HarvestSource.opensearch
OpenSearchInterface.from_environment]
  OSINIT --> IDXS[index_datasets dataset]

OpenSearch indexing call stack

flowchart TD
  IDX[index_datasets
harvester/opensearch.py] --> MAP[dataset_to_document for each Dataset]

  MAP --> XFORM[DCAT_INDEX_TRANSFORMER.transform
dataset.dcat]
  XFORM --> SPECS[DcatIndexTransformer.transform
iterates DEFAULT_FIELD_SPECS]

  SPECS --> C1[coerce_text → title, description]
  SPECS --> C2[coerce_publisher_name → publisher]
  SPECS --> C3[coerce_keywords → keyword]
  SPECS --> C4[coerce_concepts → theme
DCAT3 Concept objects kept]
  SPECS --> C5[coerce_identifier → identifier
DCAT3 Identifier @id kept]
  SPECS --> C6[distribution_titles → distribution_titles]
  SPECS --> C7[collection_reference → inSeries / isPartOf]

  MAP --> NORM[_normalize_dcat_dates
full dcat blob for nested field]
  MAP --> GEO[_geometry_centroid
dataset.translated_spatial]
  MAP --> DOC[Build OpenSearch document
_index, _id, title, slug, theme, ...]

  DOC --> BULK[helpers.streaming_bulk
opensearchpy]
  BULK --> RETRY[_run_with_timeout_retry
on ConnectionTimeout]
  RETRY --> REFRESH[_refresh indices.refresh]

Sequence: one DCAT3 dataset through the system

sequenceDiagram
  autonumber
  participant Job as harvest_job_starter
  participant HS as HarvestSource
  participant Net as download_file
  participant Rec as Record
  participant Val as dcatus3 validator
  participant DB as HarvesterDBInterface
  participant OS as OpenSearchInterface
  participant XF as DcatIndexTransformer
  participant Cluster as OpenSearch cluster

  Job->>HS: run_full_harvest()
  HS->>DB: get_latest_harvest_records_by_source()
  HS->>Net: GET data.json
  Net-->>HS: { dataset: [ DCAT3 dict, ... ] }

  loop each dataset dict
    HS->>Rec: Record(source_raw=json.dumps(sorted))
    Rec->>Rec: compare() vs internal hash
    alt unchanged
      Rec-->>HS: return early
    else create or update
      Rec->>Val: iter_errors(dcat dict)
      Val-->>Rec: valid
      Rec->>Rec: sync()
      Rec->>DB: insert_dataset / upsert_dataset
payload.dcat = DCAT3 metadata
      DB-->>Rec: Dataset ORM object
      Rec->>OS: index_datasets([dataset])
      OS->>XF: transform(dataset.dcat)
      XF-->>OS: canonical fields (identifier, theme, ...)
      OS->>OS: dataset_to_document()
      OS->>Cluster: streaming_bulk index
      Cluster-->>OS: created / updated
      OS->>Cluster: indices.refresh
    end
  end
